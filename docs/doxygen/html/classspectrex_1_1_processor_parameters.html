Detailed Description

KProcessor parameters class.

All parameters marked

Intended to be configured by the user are changeable by the user, and can be
expected to be initialized/set to a default value elsewhere after this constructor.

All parameters marked

Intended to be configured by the host are set by the host application (e.g. DAW)
and may not always be initialized/set, and probably need to be listed here in any case.
